## Module Summaries

### Task 1 — Digital Literacy Infographic
Designed an infographic on Canva titled *"Understanding Digital Literacy: Why It Matters in the Modern World"* covering digital literacy basics, safe internet practices, professional online presence, and email etiquette.

### Task 2 — Professional Platform Profiles
Created and set up profiles on GitHub, LinkedIn, Stack Overflow, and CodeChef. Includes a GitHub profile README and LinkedIn education section.

### Task 3 — Coding Practice and Google Workspace
Completed a beginner challenge on CodeChef. Created a Digital Literacy Awareness Quiz using Google Forms, linked to a Google Sheet for response tracking.

### Task 4 — Email Etiquette and Social Media
Wrote two professional emails (assignment extension request and internship application). Created a Social Media Do's and Don'ts checklist for responsible online behaviour.

### Task 5 — Cybercrime Awareness
Researched UPI payment fraud and wrote a case study based on the OLX Army officer scam pattern. Created a Stay Safe Online prevention checklist with reporting channels.
